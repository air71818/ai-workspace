# AI Workspace

A static GitHub Pages-ready portfolio mock for a modern AI research workspace.

## Features

- Chat thread with a streaming-style response animation.
- Source and file cards with a responsive context panel.
- Command palette via `Cmd/Ctrl + K`.
- Light and dark appearance toggle.
- Mobile responsive layout with no backend dependency.

## Run locally

Open `index.html` directly in a browser, or serve the folder with any static file server:

```sh
python3 -m http.server 8080
```

Then visit `http://localhost:8080`.
